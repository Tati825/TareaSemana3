# TareaSemana3
Programación Tradicional y POO en Python
Se realizan dos programas, ambos en Python, en el primero se ve la forma tradicional de programarlo con el uso de funciones; por otro lado el segundo es realizado con Programación Orientada a objetos, es decir, con clases y métodos.
En la forma tradicional se solicita los datos al usuario, el cual ingresa y luego ve reflejados estos en un mensaje en pantalla.
En POO se crean los métodos para recibir la información en el archivo mascota.py, mientras que en el main se colocan estos datos por default, obteniendo como salida algo predefinido y que se puede moldear cambiando el archivo main.py
